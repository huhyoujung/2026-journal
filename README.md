# 📓 2026 Journal

일일 회고 & AI 인사이트 분석

## 구조

```
2026-03/
├── entries/          ← 회고 작성
│   ├── 2026-03-01.md
│   └── ...
└── insights/         ← AI 분석 결과 (자동 생성)
    ├── 2026-03-01-insight.md
    └── ...
```

## 사용법

1. `YYYY-MM/entries/` 폴더에 회고 md 파일을 push
2. AI가 지금까지의 모든 회고를 분석
3. 분석 결과가 `YYYY-MM/insights/`에 자동 생성

## 인사이트 항목

- 🎭 감정/기분 변화 추이
- 🔁 반복되는 고민 & 패턴
- 🌱 성장 포인트 & 변화
- ⚠️ 개선 필요 영역
- 💡 핵심 키워드 & 관심사 변화
- 📊 전체 요약 & 트렌드
